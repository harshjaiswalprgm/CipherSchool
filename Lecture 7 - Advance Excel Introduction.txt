## Advanced Excel Formulas and Functions

### Key Formulas and Functions:
- **SUM**: Adds up a range of numbers.
- **AVERAGE**: Calculates the average value of a range of numbers.
- **MAX**: Returns the largest number in a range.
- **MIN**: Returns the smallest number in a range.
- **XLOOKUP**: Searches a range or an array and returns an item corresponding to the first match it finds.
- **SUMPRODUCT**: Multiplies corresponding elements in given arrays and returns the sum of those products.
- **OFFSET**: Returns a reference to a range that is a specified number of rows and columns from a cell or range of cells.
- **QUERY**: Allows you to run SQL-like queries on a range of data.

## Data Cleaning and Preparation Techniques

### Essential Functions for Data Cleaning:
- **TRIM**: Removes extra spaces from text.
- **PROPER**: Capitalizes the first letter of each word.
- **UPPER**: Converts text to uppercase.
- **LOWER**: Converts text to lowercase.
- **LEFT**: Extracts a specified number of characters from the start of a text string.
- **MID**: Extracts characters from the middle of a text string.
- **RIGHT**: Extracts a specified number of characters from the end of a text string.
- **CONCATENATE**: Combines multiple text strings into one.
- **TEXTJOIN**: Combines text from multiple ranges and/or strings, with a delimiter.
- **SPLIT**: Splits text into separate columns.
- **IF**: Performs a logical test and returns one value for a TRUE result and another for a FALSE result.
- **IFERROR**: Returns a value if a formula evaluates to an error; otherwise, it returns the result of the formula.

## Time and Date Manipulations

### Key Date and Time Functions:
- **DATE**: Creates a date from year, month, and day.
- **DAY**: Returns the day of the month from a date.
- **EDATE**: Returns the serial number of the date that is the indicated number of months before or after the start date.
- **TODAY**: Returns the current date.
- **NOW**: Returns the current date and time.
- **WEEKDAY**: Returns the day of the week corresponding to a date.
- **HOUR**: Returns the hour of a time value.
